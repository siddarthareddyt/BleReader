Universal Library (UL) for Android Example Projects

The UL for Android package includes example projects that you can install and run on an Android device. The source code for each example is stored in an Android project. To run an example project, you must create a new project and import the source code into your workspace.

Importing the Example Project Source Code into Eclipse

Complete the following steps to import a UL for Android example project into a workspace:

  1. Run Eclipse, choose a workspace folder, and click OK.
    To create a new folder, enter a name in the Workspace textbox.
  2. Select File»New»Project.
    The New Project dialog box opens.
  3. Expand the Android folder, select Android Project from Existing Code, and click Next.
    The Import Projects dialog box opens.
  4. Click Browse and navigate to the folder containing the example projects.
  5. Select the Copy projects into workspace checkbox.
    When this checkbox is selected, the projects are copied into the current workspace. This is useful if you want to maintain a copy of the example project that is separate from the source file. If you do not select this checkbox, any modifications to the example project are made to the source file.
  6. Click Finish.
  7. Select Window»Show View»Package Explorer to open the example in the Package Explorer.

Deploying and Running the Example Project on a Connected Device

Before you can run the example projects, you must set up the device as follows:

  • Enable USB debugging on the Android device. Refer to the device documentation for more information about how to do this.
  • If using a Bluetooth device, make sure that the device is already paired with the host Android system.

Refer to Using Hardware Devices for more information about setting up a device.

  1. From the Package Explorer, select the folder containing the project to run.
  2. Select Run»Run and choose Android Application from the Run As dialog box that opens; click OK.
    Eclipse installs the example project on the connected device; the example automatically opens when installation is complete.
  3. Tap the Detect DAQ devices button.
    A list of detected devices appears.
  4. Select a device and tap the Connect button.
  5. Tap the Start button to begin acquiring data.
  6. Tap the Stop button to stop acquiring data.
  7. Tap the Disconnect button to disconnect from the DAQ device.
    You must disconnect the DAQ device from the current example project before you can run a different example project.
